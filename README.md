## 0.背景介绍
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;这是我在大一年级做的一项大创内容。我们的设计原型是一辆麦克纳姆轮小车。我们可以通过语音控制小车，同时小车也会通过摄像头获取环境信息，做出相应动作。
